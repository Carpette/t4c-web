Sorts

« Raccourci » : appuyez ensuite sur une touche.
En jeu : appuyez sur cette touche → le curseur devient une main scintillante → cliquez votre cible. Le sort se relance ensuite automatiquement tant que vous ne faites rien d'autre (et s'arrête si la cible meurt). Échap ou clic droit pour annuler la visée.

Aide

Clic : déplacer / attaquer / ramasser / parler
Clic maintenu : déplacement continu — Flèches : déplacement clavier
Ctrl maintenu : mode combat (clic = frappe)
Touche de sort : visée (main scintillante) → clic sur la cible, puis relance automatique tant qu'aucune autre action
P : potion de vie — M : potion de mana
Molette : zoom — I/C/S/H : panneaux — Entrée : chat — Échap : menu et paramètres
Inventaire, clic droit : poser un objet au sol (échanges entre joueurs)
Chat : local (15 tuiles) par défaut — canaux mondiaux : /general, /aide, /ventes, /roleplay
Groupe : /inviter Nom (ou cliquez un joueur puis « Inviter ») — l'XP des combats est partagée à portée

☠ La mort est DÉFINITIVE. Choisissez vos combats.
